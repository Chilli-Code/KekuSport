# Code of Conduct

Este es un proyecto de software privado propiedad de **Keku Enterprise**.

El acceso al código fuente y al repositorio está restringido a 
personas autorizadas por Keku Enterprise.

Todo el personal autorizado deberá mantener la confidencialidad 
del código y la información relacionada con este proyecto.
