# KekuSport

Plataforma web oficial del torneo de fútbol organizado por **Keku Enterprise**.

> **AVISO:** Este es un software privado con ánimo de lucro. Todos los derechos reservados.

## Descripción

KekuSport es la plataforma digital del torneo de fútbol "La Copa Chontaduro", 
que incluye gestión de equipos, jugadores, tabla de posiciones, brackets de 
eliminación, módulo de apuestas ("La Porra") y más funcionalidades orientadas 
a la experiencia del torneo.

## Stack Tecnológico

- **Framework:** [Astro](https://astro.build/) v5
- **UI/Tailwind:** [Tailwind CSS](https://tailwindcss.com/) v4
- **Deploy:** [Vercel](https://vercel.com/)
- **Lenguaje:** TypeScript
